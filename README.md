# Hotel Management API

A RESTful backend application for managing hotels, rooms, and user accounts built with Go and PostgreSQL. The project follows a layered architecture 
to separate business logic from data access.

## Features

- User Registration
- User Login
- Password Hashing with bcrypt
- Hotel Management
- Room Management
- Update Room Information
- PostgreSQL Database Integration
- RESTful API Design
- Layered Architecture (Controller, Service, Repository)

## Tech Stack

- Go
- PostgreSQL
- bcrypt
- net/http
- JSON

## Architecture

- Controllers
- Services
- Repositories
- Models
